## 使用方式
- 點選`+`可篩選來源和語言😎
- 在「族語」和「中文」方塊中輸入查詢字串可跨語言查詢🥳
- 預設顯示排灣語資料，點選`語言_方言`下方的`Ｘ`可取消篩選條件🧐
- 預設顯示20筆資料，點選表格右上角的`>`可查看下一頁😁
- 互動式表格將顯示於下方，首次載入資料需要時間，請耐心等候😇
