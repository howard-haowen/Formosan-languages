>‼️此查詢系統僅供教學與研究之用，內容版權歸原始資料提供者所有。

## 使用方式
- 可點選`+`篩選來源和語言
- 可在「族語」和「中文」方塊中輸入查詢字串
- 預設顯示20筆資料，點選表格右上角的`>`可查看下一頁
